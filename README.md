# X-shortlink
![X-shortlink preview](X-shortlink.jpg)

## introduction
X-shortlink is a tool for shortening a long link into a short one or commonly abbreviated as shortlink

## Instalations
```
$ pkg update -y && pkg upgrade -y
$ apt update -y && apt upgrade -y
$ pkg install git
$ pkg install python -y
$ pip install figlet
$ git clone https://github.com/Whomrx666/X-shortlink.git
$ cd X-shortlink 
$ python X-shortlink.py
```

## Instructions
- **first**: Install tools according to the instructions above
- **second**: Enter the website link you want to shorten
- **third**: The link will be automatically shortened
- **last**: You can copy it and send it to anyone
